# Tesla Wrap Studio

Tesla Wrap Studio is an interactive Tesla wrap designer built with Next.js and HTML5 canvas rendering. Users can upload Tesla wrap templates, add custom images/designs, remove image backgrounds, reposition and scale artwork, and export a final high-quality PNG ready for Tesla customization workflows.

## Features

- Upload Tesla wrap templates
- Upload multiple custom images/designs
- Optional AI background removal
- HEIC / HEIF image support
- Drag-and-drop image positioning
- Scale and rotation controls
- Custom background colors
- Pattern overlays
- Real-time canvas rendering
- PNG export system
- Step-by-step user workflow
- Modern responsive UI

## Tech Stack

- Next.js
- React
- Tailwind CSS
- HTML5 Canvas API
- Lucide React
- @imgly/background-removal
- heic2any
